# Thirukkural

> The Tirukkuṟaḷ (Tamil: திருக்குறள், lit. 'sacred verses'), or shortly the Kural (Tamil: குறள்), is a classic Tamil language text on commoner's morality consisting of 1,330 short couplets, or kurals, of seven words each

an android app to browse [thirukkural](https://en.wikipedia.org/wiki/Kural) and add thirukkural verses on home screen as widgets

## why?
* there exists thirukkural app but none of them had widget feature
* learn jetpack compose & glance api
